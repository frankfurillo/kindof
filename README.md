# kindof 

kindof javascript util
==============

If you want something like 98 but not quite 98, kindof gives you kindof 98.
--------------
kindof will work on numbers and colors. 
more stuff will probably be added

Usage:

    <script src="kindof.js"></script>
	<script>
		alert(kindof(98));
		alert(kindof('#CCD4E7')); 
	</script>
